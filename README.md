# Dominos-Sales-Analysis
## Overview
This project focuses on analyzing Domino’s pizza sales data to understand overall business performance and customer ordering patterns. Using Excel for data preparation and Power BI for visualization, an interactive dashboard was created to track key sales metrics and trends.

## Objectives
Analyze overall sales and order performance
Identify peak sales months and order hours
Understand customer preferences by pizza type and size
Highlight top-selling pizzas and categories

## Key KPIs
Total Orders
Total Pizzas Ordered
Total Sales Revenue
Total Quantity Sold
Average Order Value (AOV)
Sales by Month
Orders by Order Hour

## Key Insights
Total sales reached $818K with 21.4K orders processed.
Peak sales occurred during specific months, indicating seasonal demand trends.
Evening hours showed the highest order volume, highlighting customer behavior patterns.
Classic and Supreme pizzas contributed the highest share of revenue.
Large-sized pizzas generated the most sales compared to other sizes.
Top-selling pizzas include Classic Deluxe and Barbecue Chicken varieties.

## Tools & Technologies
Excel (Data Cleaning & Preparation)

Dashboard Features

## KPI cards for quick performance overview
Monthly sales trend analysis

## Conclusion
This dashboard provides actionable insights into Domino’s sales performance, helping stakeholders optimize pricing, inventory, and marketing strategies using data-driven insights.
